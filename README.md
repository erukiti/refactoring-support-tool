> リファクタリング (refactoring) とは、コンピュータプログラミングにおいて、プログラムの外部から見た動作を変えずにソースコードの内部構造を整理することである。

これ、ASTを解析して動作が同一であればリファクタリングと見なせる、みたいなプログラムを書けば、リファクタリングを、ユニットテストとかなしで出来るんでは？？？？？？

ということで、そういうの作ろうとしてる感じ。
